Лекция 1 Создание главного окна приложения
22 шага
🎯 Цель работы Создать основное окно «Персональный менеджер задач» — скелет, на котором строятся все последующие модули.
✅ Ожидаемый результат Запущенное окно QMainWindow с заголовком, настраиваемым размером и базовой структурой классов.
QApplication QMainWindow Цикл событий setWindowTitle / resize setCentralWidget
  1. Создайте новую папку проекта task_manager/ на рабочем столе или в удобном месте.
  2. Внутри папки создайте файл main.py — это будет точка входа всего приложения.
  3. Откройте main.py в редакторе и добавьте импорт: from PySide6.QtWidgets import QApplication, QMainWindow, QWidget
  4. Добавьте импорт: import sys — он нужен для передачи аргументов командной строки в QApplication.
  5. Создайте класс MainWindow, унаследованный от QMainWindow: напишите class MainWindow(QMainWindow): и метод __init__(self).
  6. В методе __init__ вызовите super().__init__() — обязательная инициализация родительского класса.
  7. Установите заголовок окна: self.setWindowTitle('Менеджер задач').
  8. Задайте минимальный размер окна: self.setMinimumSize(900, 600).
  9. Установите начальный размер: self.resize(1000, 700).
  10. Создайте центральный пустой виджет: container = QWidget() и установите его: self.setCentralWidget(container).
  11. Добавьте метод _setup_ui(self) (пока пустой) — в него в будущем будет вынесена вся логика создания интерфейса.
  12. Вызовите self._setup_ui() в конце __init__.
  13. Ниже класса добавьте блок запуска: if __name__ == '__main__':.
  14. Внутри блока создайте объект приложения: app = QApplication(sys.argv).
  15. Создайте экземпляр окна: window = MainWindow().
  16. Вызовите window.show() для отображения окна.
  17. Запустите цикл событий: app.exec().
  18. Запустите приложение командой python main.py и убедитесь, что окно открывается.
  19. Проверьте, что заголовок окна соответствует заданному тексту 'Менеджер задач'.
  20. Попробуйте изменить resize на setFixedSize(1000, 700) — убедитесь, что окно стало нерастягиваемым, затем верните resize для гибкости.
  21. Добавьте строку состояния: self.statusBar().showMessage('Готово') — она пригодится в следующих работах.
  22. Добавьте в конец файла комментарий-заглушку: # TODO: добавить меню, панель инструментов, список задач.

Лекция 2 Форма добавления задачи
22 шага
🎯 Цель работы Реализовать виджеты ввода данных: текстовое поле, чекбокс срочности и выпадающий список приоритета. Связать их через сигналы и слоты.

QLabel QLineEdit QCheckBox QComboBox QPushButton clicked / textChanged connect()
  1. Создайте новый файл task_form.py в папке проекта.
  2. Импортируйте необходимые классы: from PySide6.QtWidgets import QWidget, QLabel, QLineEdit, QCheckBox, QComboBox, QPushButton, QVBoxLayout
  3. Создайте класс TaskForm(QWidget) с методом __init__(self, parent=None).
  4. В __init__ вызовите super().__init__(parent) и создайте метку заголовка: QLabel('Новая задача').
  5. Создайте текстовое поле для названия задачи: self.title_input = QLineEdit() и задайте placeholder: self.title_input.setPlaceholderText('Название задачи...').
  6. Создайте QLabel('Описание:') и поле self.desc_input = QLineEdit() с placeholder 'Краткое описание...'.
  7. Создайте выпадающий список: self.priority_combo = QComboBox() и добавьте элементы: addItems(['Низкий', 'Средний', 'Высокий', 'Критический']).
  8. Создайте чекбокс срочности: self.urgent_check = QCheckBox('Срочная задача').
  9. Создайте кнопку добавления: self.add_button = QPushButton('Добавить задачу').
  10. Сразу после создания кнопки установите её неактивной: self.add_button.setEnabled(False).
  11. Создайте метод _on_add_clicked(self) — обработчик нажатия кнопки.
  12. В _on_add_clicked получите значения: title = self.title_input.text(), priority = self.priority_combo.currentText(), urgent = self.urgent_check.isChecked().
  13. Добавьте проверку: если title пустой — выведите print('Введите название!') и выйдите из метода.
  14. Выведите собранные данные: print(f'Задача: {title}, Приоритет: {priority}, Срочная: {urgent}').
  15. Подключите сигнал кнопки к слоту: self.add_button.clicked.connect(self._on_add_clicked).
  16. Создайте слот _on_title_changed(self, text) и в нём активируйте/деактивируйте кнопку: self.add_button.setEnabled(bool(text)).
  17. Подключите: self.title_input.textChanged.connect(self._on_title_changed).
  18. Создайте слот _on_priority_changed(self, index), подключите currentIndexChanged и в слоте выводите выбранный приоритет в консоль.
  19. Создайте метод clear_form(self): self.title_input.clear(), self.desc_input.clear(), self.urgent_check.setChecked(False).
  20. Вызовите self.clear_form() в конце _on_add_clicked после успешного добавления.
  21. Соберите все виджеты в QVBoxLayout и установите его: self.setLayout(layout).
  22. В main.py импортируйте TaskForm, создайте экземпляр и временно установите как centralWidget. Запустите и проверьте кнопку.
